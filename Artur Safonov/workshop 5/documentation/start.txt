Для запуску проекта послідовно виконуються наступні дії: у папці bot запускається файл model.py для створення відповідних таблиць у БД (перед цим необхідно ввести правильні дані для підключення до БД у файлі database_connection.py). Також необхідно задати токен телеграм бота у файлі config.py. Після цього запускається бот (файл bot.py).

Для запуску сайта достатньо у папці site запустити файл main.py.
